What's a Web Browser?

Web browsers are incredibly useful software programs that allow you to search for and view various kinds of information on the Web, such as web sites, video, audio, etc., putting all this information into a more easily accessible, viewable format. In other words, rather than seeing the "ingredients" of the various websites we like to visit, a Web browser makes it possible for us to see the finished product a Web browser makes it possible for us to see the finished product.

Throughout this site you will find a wealth of information on Web browsers, including different styles of browser and tutorials on how to use them for different purposes.

Browser Features

Most browers contain the following features:

  • Address field
  • Status bar
  • Back button
  • Bookmarks/Favorites
  • Close box
  • Forward button
  • History button
  • Home button
  • Maximize button
  • Minimize button
  • Print button
  • Reload/Refresh button

URL

Web sites are found by their addresses on the World Wide Web. These addresses are known as URLs, or Uniform Resource Locators. Every website has a URL assigned to it, so both searchers and Web servers can find them quickly and easily. It consists of the following parts:

  • http://:: stands for Hypertext Transfer (or Transport) Protocol
  • domain name: textual representation of IP address, e.g. 69.63.176.13
  • forward slash: path or directory on the computer to file
  • name of file: name of file, usually ending in .html or .htm

Address bar

An address bar is a text field near the top of a Web browser window that displays the URL of the current webpage. The URL, or web address, reflects the address of the current page and automatically changes whenever you visit a new webpage. Therefore, you can always check the location of the webpage you are currently viewing with the browser's address bar.

Status bar

A status bar is the bottom partition of Internet browsers and program windows that display the current state of the web page or window being displayed. For example, in Internet Explorer, it shows whether or not the page is secure, its certificate, what is currently being loaded on the page, and the web address.
