Firefox is a Web browser that is smaller, faster, and in some ways more secure than the Mozilla browser from which much of its code was originally derived. Compared to Internet Explorer, the most popular Web browser, Firefox gives users a cleaner interface and faster download speeds. As of November 2008, Firefox had about 20% of the market share for browsers, worldwide. Internet Explorer had about 70%. Firefox includes most of the features with which users of other browsers are familiar. The first iteration of the browser also included several new features other browsers did not have at the time, such as a bookmarks toolbar and tabbed browsing that allow a user to quickly switch back and forth between several Web sites.

Live book-marks

Firefox features a very useful technology to book mark interesting websites. All you have to do is drag the tab to book marks area and the web page appears as a box there. Browser book marks are immediately saved; they will remain when you will open Firefox for the next time time.

Tabbed browsing

Browser tabs do not appear when you firstly start Firefox; they can be enabled by right clicking on any web link and selecting “Open Link in New Tab” from a contextual menu (automatically opens required link on a new tab in Firefox toolbar), or by simply clicking Ctrl + T (creates a new tab).

Find phrase

Internal search feature in Firefox web browser allows searching for a word or phrase within a web page. Search functionality is enhanced with live seek feature – Firefox automatically scrolls to the first found phrase as you are typing, if phrase is not found input bar becomes red. Firefox search can be called by clicking Ctrl + F.

Firefox UPDATES

Being an open source web browser, Firefox is constantly reviewed and updated by thousands of security experts. Using automated update system fixes are delivered to user's computer very quickly.

Phishing Protection

With help of Google, Firefox team has developed a protection against untrusted sites. If the site you're entering is suspected to be malicious, Firefox will show a message indicating that you shouldn't go further.

Auto-Install Protection

Firefox will not allow any software to be installed automatically. You will always be noticed if website requests any specific installation. If you will see a dialog asking for an unknown program, simply click “cancel”.
